Geolin Tools v1.0 — Pilih topik di bawah
📱 Apa itu Geolin Tools?
Geolin Tools adalah aplikasi web yang membantu tim marketing mengelola tiga hal utama secara otomatis melalui n8n:
💬
Follow Up Kirim pesan WA massal ke kontak berlabel via WAHA + AI Gemini
📡
Send Event Kirim event Purchase ke Facebook CAPI berdasarkan kode promo
🔍
Cari Kode Promo Temukan kode promo customer berdasarkan waktu pesan masuk
✅ Urutan Setup Awal
1
Import semua workflow n8n (Follow Up, Send Event, Lacak Waktu) ke akun n8n kamu
2
Aktifkan semua workflow di n8n — pastikan statusnya Active
3
Salin URL webhook dari setiap workflow di n8n
4
Buka halaman Pengaturan di aplikasi ini, isi semua webhook URL dan data lainnya
5
Klik Simpan Pengaturan — lalu siap digunakan!
💡 Pengaturan disimpan di browser masing-masing device. Setiap device/user perlu mengisi pengaturan sendiri.

📊 Google Spreadsheet
Digunakan khusus untuk fitur Cari Kode Promo:
1
URL Spreadsheet — tempel URL lengkap Google Sheet yang berisi data kode promo, contoh:
https://docs.google.com/spreadsheets/d/1ABC.../edit
2
ID Sheet — isi dengan ID numerik tab sheet. Cara cari: lihat URL setelah #gid=
Contoh: 0 atau 1064858876
⚠️ Pastikan akun Google Sheets di n8n sudah punya akses baca ke spreadsheet ini.
📺 Pixel Facebook
Data pixel digunakan oleh fitur Send Event:
1
ID Pixel — angka ID pixel dari Meta Business Suite
2
Nama Pixel — nama bebas untuk membedakan pixel (misal: Pixel Tokped)
3
Token Pixel — Access Token dari Meta Events Manager → Pengaturan → Generate Access Token
💡 Bisa menambahkan banyak pixel. Saat Send Event, pilih pixel mana yang ingin digunakan.

💬 Apa itu Follow Up?
Fitur ini mengirim pesan WhatsApp otomatis ke semua kontak yang memiliki label tertentu di WA. Pesan dibuat oleh AI Google Gemini dan dikirim satu per satu secara berurutan via WAHA.
📋 Cara Menggunakan
1
Buka halaman Follow Up dari menu
2
Pilih Akun WA yang akan digunakan untuk mengirim
3
Pilih Aksi (template pesan follow up yang ingin dikirim)
4
Klik Proses Sekarang — tunggu konfirmasi sukses
5
n8n akan memproses pengiriman ke semua kontak berlabel secara otomatis di background
⚠️ Jangan klik berulang kali — satu klik sudah memulai semua proses di n8n.
⚙️ Alur Proses n8n
🌐
Webhook menerima request Menerima: session, id_label, aksi
↓
🏷️
Get All Labels Ambil semua label dari WA session
↓
👥
Get Chats by Label Ambil semua kontak dengan label tersebut
↓
🔄
Loop Over Items Proses setiap kontak satu per satu
↓
🤖
Google Gemini AI Generate teks pesan follow up
↓
💬
Send via WAHA Kirim pesan + gambar ke nomor WA
⚠️ Prasyarat
✓
WAHA sudah berjalan dan session sudah login/aktif
✓
Kontak sudah diberi label di WA sebelum follow up dijalankan
✓
Workflow n8n Follow Up sudah aktif
✓
Google Gemini credentials sudah dikonfigurasi di n8n
📡 Apa itu Send Event?
Fitur ini mengirim event Purchase ke Facebook Conversions API (CAPI) berdasarkan kode promo yang dimasukkan. Data diambil dari Google Sheets dan dikirim ke Meta untuk optimasi iklan.
📋 Cara Menggunakan
1
Buka halaman Send Event dari menu
2
Pilih Pixel Tujuan yang ingin menerima event
3
Masukkan Kode Promo yang diberikan customer
4
Masukkan Nomor User (nomor HP customer)
5
Klik Proses Sekarang → konfirmasi di popup → tunggu hasil
⚙️ Alur Proses n8n
🌐
Webhook menerima data pixel_id, pixel_token, kode_promo, nomor_user
↓
📊
Cari di Google Sheets Cari kode promo di sheet data
↓
🔀
Cek duplikat Jika sudah pernah dikirim → tolak
↓
🔐
Hash data (Crypto) Nomor user di-hash SHA256 untuk privasi
↓
📤
Kirim ke Facebook CAPI HTTP request ke graph.facebook.com
↓
✅
Update Sheet Tandai data sudah dikirim di Sheets
🔔 Arti Notifikasi
✅
Event Berhasil Dikirim — Data Purchase sukses terkirim ke Facebook CAPI
⚠️
Data Sudah Ada — Kode promo ini sudah pernah dikirim, tidak perlu kirim ulang
🔍
Event Tidak Ditemukan — Kode promo tidak ada di database Google Sheets
❌
Gagal Terhubung — Cek URL webhook atau koneksi internet
🔍 Apa itu Cari Kode Promo?
Fitur ini membantu mencari kode promo yang digunakan customer berdasarkan waktu pesan masuk di WhatsApp. Berguna ketika customer tidak menyebutkan kode promonya secara langsung.
💡 Bisa beda tanggal — misalnya customer klik promo jam 23:58 tapi pesan WA masuk jam 00:02 keesokan harinya. Sistem tetap bisa menemukan kode promonya.
📋 Cara Menggunakan
1
Lihat waktu pesan WA customer masuk (jam dan tanggal)
2
Buka halaman Cari Kode Promo
3
Koreksi Waktu Pesan Masuk sesuai waktu di WA (defaultnya waktu sekarang)
4
Pilih Toleransi Waktu — gunakan ± 5 menit untuk kebanyakan kasus
5
Klik 🔍 Cari Kode Promo — hasil akan muncul di bawah
⚙️ Alur Proses n8n
🌐
Webhook menerima request waktu_input, timestamp_target, toleransi_menit, spreadsheet_url, sheet_id
↓
📊
Baca Google Sheets Ambil semua baris dari sheet yang dipilih
↓
🧮
Hitung Selisih Waktu Bandingkan timestamp setiap baris dengan waktu target ± toleransi
↓
✅
Kembalikan hasil Kode promo terdekat dalam rentang toleransi
📊 Format Data di Spreadsheet
Kolom yang wajib ada di Google Sheets:
timestamp
Waktu klik promo dalam format Unix milliseconds (angka, bukan tanggal teks)
promo
Kode promo yang digenerate saat customer klik
phone
Nomor HP customer (opsional, ditampilkan di hasil)
utm_campaign
Nama campaign iklan (opsional)
utm_content
Konten iklan (opsional)
fbclid
Facebook Click ID (opsional)
page
URL halaman yang diklik (opsional)
⚠️ Kolom timestamp dan promo wajib ada. Tanpa itu pencarian tidak akan berhasil.
🔔 Arti Hasil Pencarian
✅
Kode Promo Ditemukan — Tampil kode promo, waktu klik, selisih waktu, dan info tambahan
🔍
Tidak Ditemukan — Tidak ada data dalam rentang waktu tersebut. Coba perluas toleransi atau cek tanggalnya
⚙️
Spreadsheet belum lengkap — Isi URL Spreadsheet dan ID Sheet di Pengaturan terlebih dahulu
💡 Tips Toleransi Waktu
± 2 menit
Koneksi internet stabil, pesan masuk cepat
± 5 menit
Standar — cocok untuk kebanyakan kasus
± 10 menit
Koneksi lambat atau customer sempat menunggu
± 15 menit
Customer klik jauh sebelum kirim pesan WA

1
Login ke dashboard n8n kamu
2
Klik tombol + atau New Workflow
3
Klik ikon ⋯ (tiga titik) di kanan atas → pilih Import from File
4
Upload file .json yang tersedia
5
Klik Save lalu aktifkan dengan toggle Active di kanan atas
🔗 Cara Mendapatkan URL Webhook
1
Buka workflow yang sudah diimport
2
Klik node Webhook (node pertama berwarna hijau/oranye)
3
Salin URL yang tertera di bagian Webhook URL (gunakan URL Production)
4
Tempel URL tersebut di halaman Pengaturan sesuai fungsinya
⚠️ Gunakan Production URL (bukan Test URL) agar webhook berjalan saat workflow aktif tanpa harus buka n8n.

🚨 Troubleshooting
❌
Webhook tidak merespons — Pastikan workflow n8n sudah Active dan URL webhook benar
❌
Google Sheets error — Cek credentials Google Sheets di n8n dan akses sheet sudah benar
⚠️
Pesan WA tidak terkirim — Cek session WAHA apakah masih aktif dan terhubung
⚠️
Cari promo selalu kosong — Pastikan kolom timestamp di Sheets berisi angka Unix milliseconds, bukan format tanggal
